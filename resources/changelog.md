# Changelog

## 0.1.0 — 2026-06-30

Initial version of the QuadX HRIS Design System documentation site.

**Foundations**
- Color primitives (8 brand + 11 neutral + 8 status + 5 surface aliases)
- Typography scale (10 size steps, 3 line-heights, named text styles)
- Spacing scale (10 steps, layout constants, utility classes)
- Radius scale (5 steps)
- Shadow scale (4 levels)

**Components — Shared (@xhr/ui)**
- Banner, Button, Card, Chip, DataTable, FormField, Input, ListRow, Modal, PageHeader, Select, StatTile, Textarea

**Components — Custom**
- AppShell, ClockCard, CorrectionModal, EmpPhoto, LeaveModal

**Patterns**
- ApprovalsInbox, FilteredTable, FormModal, OrgChart, RtoCard, WeekGrid

**Resources**
- Traceability table cross-referencing source, Figma, and web DS
- Known gaps documented

**Figma library:** HRIS-DS (Refinement Pass 1 complete — 18 components + 5 patterns + Icons page)
